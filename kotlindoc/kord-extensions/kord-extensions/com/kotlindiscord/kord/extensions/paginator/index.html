Paginator

class Paginator

Interactive embed with multiple pages using emoji reactions as user inputs.

NOTE : .send() needs to be called in order for the paginator to be displayed.

Fields

Name Description
var message: Message?

Message containing the embed.
var currentPage: Int

Current page of the paginator.
var doesProcessEvents: Boolean

Whether the paginator still processes reaction events.
val bot: ExtensibleBot

Current instance of the bot.
val channel: MessageChannelBehavior

Channel to send the embed to.
val name: String

Title of the embed.
val pages: List<String>

List of the embed pages.
val owner: User?

Only user capable of interacting with the paginator. Everyone is able to if it is set to null.
val timeout: Long

Milliseconds before automatically deleting the embed. Set to a negative value to disable it.
val keepEmbed: Boolean

Keep the embed and only remove the reaction when the paginator is destroyed.

suspend fun destroy()

Destroy the paginator.

This will make it stops receive ReactionAddEvent and will delete the embed if keepEmbed is set to true, or will delete all the reactions if it is set to false.
