parseMessage

fun parseMessage(message: Message): Array<String>

Takes a Message object and parses it using a StringTokenizer.

This tokenizes a string, splitting it into an array of strings using whitespace as a delimiter, but supporting quoted tokens (strings between quotes are treated as individual arguments).

This is used to create an array of arguments for a command's input.
